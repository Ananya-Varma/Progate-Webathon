HTML & CSS

Languages used to build and design the appearances of your websites. HTML adds the structure, and CSS styles it.

JavaScript

JavaScript enables interactive web pages and is an essential part of web applications. The vast majority of websites use it for client-side page behavior.

C/C++

C is one of the easiest and best languages to start programming with. C++ is an extension of C, with many added functionalities.

Python

Python is dynamically typed and garbage-collected. Supports procedural, object-oriented, and functional programming.

Java

Java is a general-purpose programming language that is class-based, object-oriented, and designed to have as few implementation dependencies as possible.

PHP

PHP is a popular general-purpose scripting language that is especially suited to web development.
